1 수집하는 개인정보

✅

체크메이트는 회원가입, 로그인, 개인정보 입력을 요구하지 않습니다.

앱은 사용자의 기기 내부(로컬 스토리지)에만 데이터를 저장하며, 외부 서버로 어떠한 개인 데이터도 전송하지 않습니다.

  • 수입·지출 내역, 예산, 자산 정보 — 기기 내 저장, 외부 전송 없음
  • 앱 사용 통계 — 수집하지 않음
  • 위치 정보 — 수집하지 않음
  • 연락처, 사진 등 민감 권한 — 요청하지 않음

2 데이터 저장 방식

입력한 모든 데이터는 사용자의 기기(브라우저 로컬 스토리지 또는 앱 내부 저장소)에만 저장됩니다.

📱 데이터는 오직 사용자의 기기에만 존재합니다. 개발자도 접근할 수 없습니다.

앱 삭제 시 모든 데이터가 영구 삭제됩니다. 별도의 백업 기능을 사용하지 않는 한 복구가 불가능하므로 주의하세요.

3 제3자 제공 및 공유

체크메이트는 사용자의 데이터를 제3자에게 제공하거나 판매하지 않습니다.

  • 광고 네트워크에 데이터 제공 — 없음
  • 분석 서비스(Google Analytics 등) 연동 — 없음
  • 외부 API 호출 — 없음

4 인터넷 연결 및 권한

앱 설치 후 오프라인에서도 완전히 사용 가능합니다. 다음 목적으로만 인터넷 연결이 사용됩니다.

  • 최초 앱 로드 시 폰트·리소스 캐시 다운로드
  • 앱 업데이트 확인

그 외 민감 권한(카메라, 마이크, 위치, 연락처 등)은 요청하지 않습니다.

5 아동 개인정보 보호

체크메이트는 만 14세 미만 아동을 대상으로 하지 않으며, 아동으로부터 고의로 개인정보를 수집하지 않습니다. 아동이 앱을 사용하더라도 개인정보가 외부로 전송되지 않습니다.

6 방침 변경 안내

개인정보처리방침이 변경될 경우, 앱 내 공지 또는 본 페이지를 통해 사전에 안내드립니다. 변경된 방침은 공지 후 7일이 경과한 시점부터 효력이 발생합니다.

지속적인 앱 사용은 변경된 방침에 동의하는 것으로 간주됩니다.
